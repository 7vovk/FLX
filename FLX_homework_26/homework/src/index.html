Keyboard shortcuts

  1. "c" - clear all
  2. "backspace" - clear last symbol
  3. "enter" - count expression
  4. "_" (shift and -) - add "-" sign before number (- the key at the top of keyboard in the number line)
  5. "s" - take square from entered number
  6. "%" (shift and 5) - to take % from entered number
  7. "m" - add number to memory
  8. "ctrl+m" - get number from memory and clear memory
  9. "+,-,*,/" - for +, -, *, /
